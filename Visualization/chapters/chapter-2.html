Chapter 2

Visual Mappings

How raw data is transformed into visual forms that people can understand.

What is Visual Mapping?

Visual mapping is the process of transforming raw data into a visual form. It answers the central design question: How do we show the data?

The InfoVis Pipeline

Source Data→ Data Tables→ Visual Abstractions→ Views
  • Source data: raw data before processing.
  • Data tables: structured data, usually organized into rows and columns.
  • Visual abstraction: the mapping of data to visual elements.
  • Views: the final visual representation shown to the user.

The pipeline explains how a visualization is constructed technically, but does not fully describe how people interpret and make sense of data.

Main Components of Visualization

  • Spatial substrate: the space in which the visualization is drawn.
  • Marks: visual objects such as points, lines, areas, and volumes.
  • Graphical properties: position, color, shape, size, texture, and orientation.

Spatial Substrate

The spatial substrate is the available visual space. Because screen pixels are limited, designers must use this space efficiently to reveal relationships, comparisons, patterns, and structures.

Marks

Marks are the basic objects that communicate information:

  • Points
  • Lines
  • Areas
  • Volumes

Graphical Properties

Graphical properties control how marks appear and encode differences in the data. They include position, color, shape, size, texture, and orientation.

Basic Types of Visualizations

1. Data Tables

Tables organize data into rows and columns. Rows usually represent cases, while columns represent variables.

2. Graphs on Rails and Planes

Graphs on rails place values along one-dimensional positions or axes. Graphs on planes use both an x-axis and a y-axis, allowing two variables to be compared simultaneously.

3. Geospatial Graphs

Geospatial graphs use position to represent meaningful physical space. A major challenge is projecting the round Earth onto a flat surface without excessive distortion.

4. Network Diagrams and Graphs

Network diagrams emphasize relationships and connectivity. Exact geographic position may be simplifiedâ€”as in subway mapsâ€”to make connections easier to understand.

5. Conceptual Symbols and Text

Symbols and text can form part of a larger visual structure, communicating an idea collectively rather than as isolated elements.

Infographics

Infographics combine charts, maps, symbols, text, and diagrams. Their primary purpose is usually clear communication and explanation rather than deep data exploration.

Visualizations by Number of Variables

Univariate Data

Univariate data contains one main variable. Common representations include dot plots, box plots, line plots, and histograms. They reveal distributions, trends, and outliers.

Bivariate Data

Bivariate data contains two variables. A scatterplot maps each case to a pair of coordinates:

(x, y)

Scatterplots reveal relationships, correlations, clusters, and outliers. For temporal data, time commonly appears on the horizontal axis.

Trivariate Data

Trivariate data contains three variables. Instead of a difficult-to-read 3D plot, a 2D layout can encode the third variable with color, size, or shape.

x→ horizontal position
y→ vertical position
z→ color, size, or shape

The data can also be divided into several separate 2D views.

Hypervariate Data

Hypervariate data contains many variables and is difficult to represent directly because people perceive only a limited number of spatial dimensions.

  • Multiple views: separate displays for different dimensions.
  • Small multiples: repeated charts that support comparison.
  • Scatterplot matrix: a scatterplot for every pair of variables.
  • Table lens: a table-like representation of many dimensions.
  • Parallel coordinates: each data case becomes a line crossing several vertical axes.

Parallel Coordinates

Each variable is represented by a vertical axis and every data item becomes a line crossing all axes. Variables with different ranges normally need to be normalized:

x′ = (x − xmin) / (xmax − xmin)
x′ ∈ [0, 1]

Normalization makes variables with different units and ranges comparable on the same visual scale.

Other Ways to Classify Data

  • Temporal data: data involving time.
  • Geospatial data: data involving physical or geographic space.
  • Relational data: data involving connections or networks.
  • Textual data: data based on words, documents, or language.

Chapter Takeaway

Visual mappings connect data to visual representations through space, marks, and graphical properties. Choosing an effective mapping depends on the nature of the data, the number of variables, and the user's task.
