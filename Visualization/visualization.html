Exam method

How to Critique a Visualization

Critique questions ask: what is wrong with this visualization, why is it wrong, and how can we fix it?

Use This Method Every Time

Task first. Encoding second. Perception third. Fix last.

1. Start With The Task

Ask: what is the user supposed to do?

compare values see trends find clusters find outliers understand hierarchy compare categories follow change over time

If the visualization does not help the task, that is your critique.

2. Check The Encoding

Ask: are the visual channels good? Best channels for accurate comparison:

  1. position on common scale
  2. length
  3. angle/slope
  4. area
  5. color

If a chart uses area, 3D, color, or angle when the task is precise comparison, criticize it.

The visualization makes comparison difficult because it uses area/color instead of position or length. A bar chart or aligned dot plot would be more effective.

3. Check Perception Problems

too many colors rainbow color scale low contrast clutter overlapping labels 3D distortion bad ordering missing scale misleading axis chartjunk

The rainbow scale is problematic because the colors are not perceptually ordered, so users may misread high and low values.

4. Check If It Lies

  • Is the axis truncated?
  • Are sizes proportional to the data?
  • Does the visual effect exaggerate the real effect?
  • Is context missing?

This is Tufte-style critique.

The chart exaggerates the difference because the y-axis does not start at zero, increasing the perceived change.

5. Always Give A Fix

Never only say what is wrong. Add the better design.

A better solution would be to use ___ because the task is ___.

A better solution would be to use an adjacency matrix because the graph is dense and matrix views avoid edge clutter.

Exam Template

The main problem is that the visualization does not support the task of ____ well.

It uses ____ encoding, which makes ____ difficult because ____.

There is also a perception/design issue: ____.

A better solution would be ____ because it would make ____ easier to see/compare/understand.
