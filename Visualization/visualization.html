How to Critique a Visualization

Critique questions ask: what is wrong with this visualization, why is it wrong, and how can we fix it?

Task first. Encoding second. Perception third. Fix last.

1. Start With The Task

Ask: what is the user supposed to do?

compare values see trends find clusters find outliers understand hierarchy compare categories follow change over time

If the visualization does not help the task, that is your critique.

2. Check The Encoding

Ask yourself: how is the data represented visually?

In a visualization, data is shown using visual channels:

Data shown byEncoding channel
x/y positionposition
bar heightlength
circle sizearea
color brightnessluminance
color categoryhue
line thicknesswidth
symbol shapeshape
connection between nodesedge/link

The critique is: is this a good channel for the task?

Example 1: Exact Values

If the task is to compare exact values, using color is weak.

Bad: Values are encoded with color intensity, so comparing exact numbers is difficult.

Better: Use bars or aligned dots, because position/length is easier to compare.

Example 2: Categories

If the task is to show categories, color is good.

Good: Different categories are encoded by different colors, which is appropriate because color hue helps distinguish identity.

Example 3: Quantities With Area

If the task is to compare quantities, circle area is risky.

Bad: The values are encoded by circle area, but humans do not compare areas accurately.

Better: Use bar length or position on a common scale.

Very Simple Exam Sentence

The encoding is not effective because it uses ____ to show ____.
This makes ____ difficult.
A better encoding would be ____ because it supports the task of ____.
The encoding is not effective because it uses color to show quantitative values.
This makes accurate comparison difficult.
A better encoding would be position or length because it supports comparing values more clearly.

3. Check Perception Problems

Ask yourself: can the human eye understand this visualization easily, or is something making it hard to read?

This is about how people perceive visuals.

ProblemMeaning
too many colorsthe viewer gets confused
rainbow color scalecolors do not have a clear order
low contrasthard to separate foreground/background
cluttertoo much information at once
overlapping labelstext becomes unreadable
3D distortionshapes look bigger/smaller than reality
bad orderingcategories are not sorted logically
missing scaleuser cannot know exact values
misleading axisaxis exaggerates or hides differences
chartjunkdecorative elements that do not help understand data

Very Simple Exam Sentence

The visualization has a perception problem because ____.
This makes it difficult for the viewer to ____.
A better design would be ____.

Example

The visualization has a perception problem because there are too many colors.
This makes it difficult for the viewer to identify the important patterns.
A better design would be to use fewer colors and reserve strong colors for important values.

Another Example

The visualization uses a rainbow color scale, which is not perceptually ordered.
This makes high and low values hard to interpret.
A better design would be to use a sequential color scale from light to dark.

If the eye struggles, it is a perception problem.

4. Check If It Lies

  • Is the axis truncated?
  • Are sizes proportional to the data?
  • Does the visual effect exaggerate the real effect?
  • Is context missing?

This is Tufte-style critique.

The chart exaggerates the difference because the y-axis does not start at zero, increasing the perceived change.

5. Always Give A Fix

Never only say what is wrong. Add the better design.

A better solution would be to use ___ because the task is ___.

A better solution would be to use an adjacency matrix because the graph is dense and matrix views avoid edge clutter.

Exam Template

The main problem is that the visualization does not support the task of ____ well.

It uses ____ encoding, which makes ____ difficult because ____.

There is also a perception/design issue: ____.

A better solution would be ____ because it would make ____ easier to see/compare/understand.
